Getting Started

Get up and running with Vizient Components for web

Step 1. Download assets

Download css, javascript and fonts file

  • Download css file
  • Download javascript files (jquery & bootstrap js files are required)
  • Download fonts files and place them in the same folder with the css file)

Step 2. Include Stylesheet

Put these code in the <head>tag

<link href="font-awesome.css"/>
<link href="vizient-ux-toolkit.css"/>

Step 3. Include Javascript

put the following script before the closing <body>tag

<script src="jquery.min.js">
<script src="bootstrap.min.js">
<script src="components.js"/>
