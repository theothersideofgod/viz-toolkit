Slide Toggle
Slider Toggles are used for binary actions. They are commonly used for “On/Off” situations.
Examples Examples
Basic slide toggle
Default slide toggle is an on/off control that can be toggled via clicking or dragging.
Configurable slide toggle
Use this configuration below to customize a slide toggle that fit your needs best:
Props Props
To get a customized slide toggle, just set ariaLabel / ariaLabelledby / checked / color / disableRipple / disabled / id / labelPosition / name / required / change / dragChange / toggleChange / defaults / inputId / checked / source / disableDragValue / disableToggleValue
property description type default
ariaLabel Used to set the aria-label attribute on the underlying input element. string | null
ariaLabelledby Used to set the aria-labelledby attribute on the underlying input element. string | null
checked Whether the slide toggle element is checked or not. boolean
color Theme color palette for the component. ThemePalette
disableRipple Whether ripples are disabled. boolean
disabled Whether the component is disabled. boolean
id A unique id for the slide toggle input. If none is supplied, it will be auto-generated. string
labelPosition Whether the label should appear after or before the slide toggle. Defaults to 'after'. 'before' | 'after'
name Name value will be applied to the input element if present. string | null
required Whether the slide toggle is required. boolean
change An event will be dispatched each time the slide toggle changes its value. EventEmitter
dragChange An event will be dispatched each time the slide toggle is dragged. This event is always emitted when the user drags the slide toggle to make a change greater than 50%. It does not mean the slide toggle's value is changed. The event is not emitted when the user toggles the slide toggle to change its value. EventEmitter
toggleChange An event will be dispatched each time the slide toggle input is toggled. This event is always emitted when the user toggles the slide toggle, but this does not mean the slide toggle's value has changed. The event does not fire when the user drags to change the slide toggle value. EventEmitter
defaults MatSlideToggleDefaultOptions
inputId Returns the unique id for the visual hidden input. string
checked The new checked value of the MatSlideToggle. boolean
source The source MatSlideToggle of the event. MatSlideToggle
disableDragValue Whether drag action triggers value changes in slide toggle. boolean
disableToggleValue Whether toggle action triggers value changes in slide toggle. boolean
