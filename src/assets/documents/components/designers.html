Working with UI toolkit as a Sketch library
Sketch library makes working across multiple files easy. To have access to all of the ‘Toolkit Material Library’ symbols, you'll need to load the files as a Sketch libraries.
  • Open Sketch > Preferences > Libraries > Add Library
  • Navigate to find and select the ‘Toolkit Material Library’ Sketch file
  • Now you will be able to access all of the icons from any Sketch document from the insert menu.
Recieving library updates

To recieve updates to the Toolkit Material Library symbols make sure you periodically download the latest kits from the Vizient styleguide. Once you update your local copy with the latest version you will automatically recieve symbol updates inside of Sketch.

Updates are indicated by a badge that Sketch displays in the top right hand corner. You’ll be able to see all of the symbols that have been updated and select which ones you want to recieve at any given time.
