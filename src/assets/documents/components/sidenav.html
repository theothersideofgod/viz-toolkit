Sidenav
The sidenav components are designed to provide access to destinations in your app. They can either be permanently on-screen or controlled by a navigation menu icon.

autosize Whether to automatically resize the container whenever the size of any of its drawers changes. Use at your own risk! Enabling this option can cause layout thrashing by measuring the drawers on every change detection cycle. Can be configured globally via the MAT_DRAWER_DEFAULT_AUTOSIZE token. boolean
hasBackdrop Whether the drawer container should have a backdrop while one of the sidenavs is open. If explicitly set to true, the backdrop will be enabled for drawers in the side mode as well. any
backdropClick Event emitted when the drawer backdrop is clicked. EventEmitter
end The drawer child with the end position. MatDrawer | null
