Best Practices Best Practices

Dividers separate content into clear groups.

  • Subtle: Dividers should be noticeable in a layout, but not jarring.
  • Secondary: Dividers should only be used if elements cannot be separated using white space.
  • Infrequent: Use dividers sparingly, to create groupings rather than separate items.
