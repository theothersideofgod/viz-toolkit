Form Controls
Datepicker
Datepicker allow the user to enter a date or date range either through text input, or by choosing from the calendar.
Example
Use in forms to let user enter a date or date range.
Props
To get a customized checkbox, just set type/state/disabled
property description type default
checked Specifies whether the checkbox is selected boolean false
disabled disabled state of checkbox boolean false
Best Practice
Datepicker should:
  • Allowe user to input a date or date range.
  • Be inputed through text input, or by choosing date from calendar.
  • Always display current date.
Usage & Content guidelines
Always disable the date that user cannot interact with.
