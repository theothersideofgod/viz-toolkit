Tooltip
Tooltips are text labels that appear when the user hovers over, focuses on, or touches an element. A tooltip is displayed upon tapping and holding a screen element or component (on mobile) or hovering over it (desktop). Continuously display the tooltip as long as the user long-presses or hovers over the element.
Examples Examples
The tooltip can be displayed above, below, left, or right of the element. By default the position will be below.
