Props Props
To get a customized radio, just set disabled / labelPosition / name / required / selected / value / change
property description type default
disabled Whether the radio group is disabled boolean
labelPosition Whether the labels should appear after or before the radio-buttons. Defaults to 'after' 'before' | 'after'
name Name of the radio button group. All radio buttons inside this group will use this name. string
required Whether the radio group is required boolean
selected The currently selected radio button. If set to a new radio button, the radio group value will be updated to match the new selected button. MatRadioButton
value Value for the radio-group. Should equal the value of the selected radio button if there is a corresponding radio button with a matching value. If there is not such a corresponding radio button, this value persists to be applied in case a new radio button is added with a matching value. any
change Event emitted when the group value changes. Change events are only emitted when the value changes due to user interaction with a radio button (the same behavior as <input type-"radio">). EventEmitter
