Button Toggle
Button toggles are on/off toggles with the appearance of a button. These toggles can be configured to behave as either radio-buttons or checkboxes.

Props Props
To get a customized button-toggle, just set appearance / disabled / multiple / name / value / vertical / change / selected
property description type default
appearance The appearance for all the buttons in the group. MatButtonToggleAppearance
disabled Whether multiple button toggle group is disabled. boolean
multiple Whether multiple button toggles can be selected. boolean
name name attribute for the underlying input element. string
value Value of the toggle group. any
vertical Whether the toggle group is vertical. boolean
change Event emitted when the group's value changes. EventEmitter
selected Selected button toggles in the group.
Usage & Content Guidelines Content Guidelines
Only one item can be selected from a button toggle.
