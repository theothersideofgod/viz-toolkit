Best Practices Best Practices
Use radio button to:
  • Scannable: It should be visible at a glance if a selection has been selected, and selected items should be more visually prominent than unselected items.
  • Contextual: Select content should be suited to user needs.
  • Nimble: Selects should be easy to open, close, and interact with.
Usage & Content Guidelines Content Guidelines
Select menu should be short, clear, and fully visible. It shouldn’t be truncated.
