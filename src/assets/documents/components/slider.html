Use this configuration below to customize a slider that fit your needs best:
Props Props
To get a customized slider, just set color / disabled / displayWith / invert / max / min / step / thumbLabel / tickInterval / value / vertical / change / input / displayValue / onTouched / percent / source / value
property description type default
checked Specifies whether the checkbox is selected boolean false
color Theme color palette for the component. ThemePalette
disabled Whether the component is disabled. boolean
displayWith Function that will be used to format the value before it is displayed in the thumb label. Can be used to format very large number in order for them to fit into the slider thumb. (value: number | null) => string | number
invert Whether the slider is inverted. boolean
max The maximum value that the slider can have. number
min The minimum value that the slider can have. number
step The values at which the thumb will snap. number
thumbLabel Whether or not to show the thumb label. boolean
tickInterval How often to show ticks. Relative to the step so that a tick always appears on a step. Ex: Tick interval of 4 with a step of 3 will draw a tick every 4 steps (every 12 values). number | "auto"
value Value of the slider. number | null
vertical Whether the slider is vertical. boolean
change Event emitted when the slider value has changed. EventEmitter
input Event emitted when the slider thumb moves. EventEmitter
displayValue The value to be used for display purposes. string | number
onTouched onTouch function registered via registerOnTouch (ControlValueAccessor). () => any
percent The percentage of the slider that coincides with the value. number
source The MatSlider that changed. MatSlider
value The new value of the source slider. number | null
Usage & Content Guidelines Content Guidelines
Do not use sliders for ranges that are too large (1-1000) or too small (1-3)
