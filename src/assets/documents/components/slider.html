Slider
Sliders allow users to make selections from a range of values.
Examples Examples
Sliders reflect a range of values along a bar, from which users may select a single value. Sliders can have icons on both ends of the bar that reflect a range of values.

Best Practices Best Practices
Use slider to:
  • Illustrate/communicate a sliding scale of an option.
  • Validate input as soon as the user has finished interacting with a field.
Usage & Content guidelines Content guidelines
Do not use sliders for ranges that are too large (1-1000) or too small (1-3)
