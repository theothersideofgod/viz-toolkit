property description type default
displayMode Display mode used for all expansion panels in the accordion. Currently two display modes exist: default - a gutter-like spacing is placed around any expanded panel, placing the expanded panel at a different elevation from the rest of the accordion. flat - no spacing is placed around expanded panels, showing all panels at the same elevation. MatAccordionDisplayMode
hideToggle Whether the expansion indicator should be hidden. boolean
multi Whether the accordion should allow multiple expanded accordion items simultaneously. boolean
id A readonly id value to use for unique selection coordination. `cdk-accordion-${nextId++}`
disabled Whether the AccordionItem is disabled. any
expanded Whether the AccordionItem is expanded. any
