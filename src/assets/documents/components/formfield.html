Form Field
Form field is a component used to wrap several Angular Material components and apply common text field styles such as the underline, floating label, and hint messages.

appearance The form-field appearance style. MatFormFieldAppearance
color Theme color palette for the component. ThemePalette
floatLabel Whether the label should always float, never float or float as the user types. Note: only the legacy appearance supports the never option. never was originally added as a way to make the floating label emulate the behavior of a standard input placeholder. However the form field now supports both floating labels and placeholders. Therefore in the non-legacy appearances the never option has been disabled in favor of just using the placeholder. FloatLabelType
hideRequiredMarker Whether the required marker should be hidden. boolean
hintLabel Text for the form field hint. string
underlineRef ElementRef
align Whether to align the hint label at the start or end of the line. 'start' | 'end'
id Unique ID for the hint. Used for the aria-describedby on the form field control. string
autofilled Whether the input is currently in an autofilled state. If property is not present on the control it is assumed to be false. boolean
controlType An optional name for the control type that can be used to distinguish mat-form-field elements based on their control type. The form field will add a class, mat-form-field-type-{{ controlType }} to its root element. string
disabled Whether the control is disabled. boolean
empty Whether the control is empty. boolean
errorState Whether the control is in an error state. boolean
focused Whether the control is focused. boolean
id The element ID for this control. string
ngControl Gets the NgControl for this control. NgControl | null
placeholder The placeholder for this control. string
required Whether the control is required. boolean
shouldLabelFloat Whether the MatFormField label should try to float. boolean
stateChanges Stream that emits whenever the state of the control changes such that the parent MatFormField needs to run change detection. Observable
value The value of the control. T | null
