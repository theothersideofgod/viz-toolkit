Best Practices Best Practices
Login should:
  • Be clearly labeled so it’s obvious to the users what they need to provide to be logged in.
  • Only ask for information that’s really needed.
  • Validate input as soon as user has finished interacting with a field.
Usage & Content Guidelines Content Guidelines
Label text should be short, clear, and fully visible. It shouldn’t be truncated.
Disable confirming actions (1) until all the fields are filled.
