Iconography
Iconography uses symbols to represent objects, commands or common actions visually. They should be simple, bold enough to grab attention, and leave a lasting impression.
Icon Fonts Icon Fonts

The approved icon set is two typefaces called Font Awesome and Material Icon Font , which provides flexible, scalable vector icons that can be plugged into any application without taxing the browser with a multitude of icon images.

To search through all the icons, have a look at Font Awesome page and Material Icon page .

Sizes Sizes
Icons are auto-scaled to provide 2 sizes: normal (18px) and large (24px)

Best Practice Best Practices
  • All navigation related icons should always be solid.
  • When scaling down icons, smaller elements like circles should be solid to help with legibility.
  • When using icons as actions, all touch targets need to be 40px or higher.
