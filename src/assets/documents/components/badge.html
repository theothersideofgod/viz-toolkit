Badge
Badges are small status descriptors for UI elements. A badge consists of a small circle, typically containing a number or other short set of characters, that appears in proximity to another object.

Props Props
To get a customized badge, just set color / content / description / disabled / hidden / overlap / position / size
property description type default
color The color of the badge. Can be primary, accent, or warn. ThemePalette
content The content for the badge string
description Message used to describe the decorated element via aria-describedby string
disabled Whether the component is disabled. boolean
hidden Whether the badge is hidden. boolean
overlap Whether the badge should overlap its contents or not boolean
position Position the badge should reside. Accepts any combination of 'above'|'below' and 'before'|'after' MatBadgePosition
size Size of the badge. Can be 'small', 'medium', or 'large'. MatBadgeSize
Usage & Content Guidelines Content Guidelines
Badge should not block the over elements it support. Bagde should always stay on the right of the main elements.
Do not add shadow to the badge.
Do not use the same color as the main elements.
