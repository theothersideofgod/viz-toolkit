Stepper
Steppers provides a wizard-like workflow by dividing content into logical steps. Steppers display progress through a sequence of logical and numbered steps. They may also be used for navigation.

Props Props
To get a customized stepper, just set linear / selected / selectedIndex / animationDone / selectionChange / steps
property description type default
linear Whether the validity of previous steps should be checked or not. boolean
selected The step that is selected. CdkStep
selectedIndex The index of the selected step. number
animationDone Event emitted when the current step is done transitioning in. EventEmitter
selectionChange Event emitted when the selected step has changed. EventEmitter
steps The list of step components that the stepper is holding. QueryList
labelPosition Whether the label should display in bottom or end position. 'bottom' | 'end'
Usage & Content Guidelines Content Guidelines
Don’t embed steppers within steppers or use multiple steppers on one page.
