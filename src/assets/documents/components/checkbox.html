Checkbox
Checkboxes allow the user to select one or more items from a set. Checkboxes can be used to turn an option on or off.
Examples Examples
Basic checkboxes
Checkboxes are for making selections from a list. They let users select more than one item and allow for easy selection or deselection of all items with a parent checkbox.
Configurable checkbox

property description type default
ariaLabel Attached to the aria-label attribute of the host element. In most cases, arial-labelledby will take precedence so this may be omitted. string
ariaLabelledby Users can specify the aria-labelledby attribute which will be forwarded to the input element string | null
checked Whether the checkbox is checked. boolean
color Theme color palette for the component. ThemePalette
disableRipple Whether ripples are disabled. boolean
disabled Whether the checkbox is disabled. This fully overrides the implementation provided by mixinDisabled, but the mixin is still required because mixinTabIndex requires it. any
id A unique id for the checkbox input. If none is supplied, it will be auto-generated. string
indeterminate Whether the checkbox is indeterminate. This is also known as "mixed" mode and can be used to represent a checkbox with three states, e.g. a checkbox that represents a nested list of checkable items. Note that whenever checkbox is manually clicked, indeterminate is immediately set to false. boolean
labelPosition Whether the label should appear after or before the checkbox. Defaults to 'after' 'before' | 'after'
name Name value will be applied to the input element if present string | null
required Whether the checkbox is required. boolean
value The value attribute of the native input element string
change Event emitted when the checkbox's checked value changes. EventEmitter
indeterminateChange Event emitted when the checkbox's indeterminate value changes. EventEmitter
inputId Returns the unique id for the visual hidden input. string
ripple Reference to the ripple instance of the checkbox. MatRipple
checked The new checked value of the checkbox. boolean
source The source MatCheckbox of the event. MatCheckbox
