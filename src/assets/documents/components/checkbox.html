Form Control
Checkbox
Checkboxes allow the user to select one or more items from a set. Checkboxes can be used to turn an option on or off.
Example
Use in forms to toggle the state of something on or off.
Props
To get a customized checkbox, just set type/state/disabled
property description type default
checked Specifies whether the checkbox is selected boolean false
disabled disabled state of checkbox boolean false
Best Practice
Use checkboxes to:
  • Select one or multiple items from a list
  • Present a list containing sub-selections
  • Turn an option on or off
Usage & Content guidelines
Start with a capital letter
Avoid using button toggles for a list of options
