Checkbox
Checkboxes allow the user to select one or more items from a set. Checkboxes can be used to turn an option on or off.
Examples Examples
Checkboxes are for making selections from a list. They let users select more than one item and allow for easy selection or deselection of all items with a parent checkbox.
When not all child checkboxes are checked, the parent checkbox becomes an indeterminate checkbox.
When checkbox is disabled, actions are unusable and un-clickable, it only serves the purpose of a visual representation of the data.

Best Practices Best Practices
Use checkboxes to:
  • Select one or multiple items from a list
  • Present a list containing sub-selections
  • Turn an option on or off
